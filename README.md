# Customer Segmentation using K-Means Clustering

## Objective

Segment customers based on income and spending behavior using clustering techniques.

## Dataset

Mall Customer Dataset from kaggle

## Tasks Performed

* Data preprocessing
* Feature selection
* K-Means clustering
* Cluster visualization
* Customer segment analysis

## Technologies Used

* Python
* Pandas
* Matplotlib
* Scikit-Learn

```bash
pip install pandas matplotlib scikit-learn numpy
```

## Run

```bash
python task6.py
```

## Output

* Customer clusters
* Cluster visualization
* Segment insights
